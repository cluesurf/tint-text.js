  Isomorphic Color Logging in TypeScript
</p>

<br/>
<br/>
<br/>

## Installation

```
pnpm add @tunebond/tint-text
yarn add @tunebond/tint-text
npm i @tunebond/tint-text
```

## Usage

```ts
import tint from '@tunebond/tint-text'

const P = { tone: '#cc9aff' }
const G = { tone: '#8dd484' }

console.log(`${tint('hello', P)} ${tint('world', P)}`)
```
